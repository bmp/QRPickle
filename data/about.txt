QRPPickle

This is a custom-built, open-source dashboard and environmental monitor for mateur radio operators, utilizing the ESP32 platform.

It is a highly opinionated take on a HAM Clock, with QRP field operations being the first-class citizen use-case.

Features:
- Live Local and UTC Time tracking
- Native Web Configuration Portal
- Live Wi-Fi & NTP Synchronization
- Live Wx data (BME280 I2C Sensor & OpenWeatherMap API)
- Forecast Wx data from OpenWeatherMap API
- DX Clusters using Telnet
- xOTA spots (currently has POTA and SOTA) using HTTP
- Band conditions and Solar propogation
- HamAlert triggers via Telnet
- APRS-IS, APRS Wx transmit and APRS messaging using Telnet
- Spotting through AP2SOTA and POTAGW from APRS
- OTA Updates
- Turning off screen with a timer
- Profiles for easier switching in the field

Created by VU3GLJ with LLMs.
